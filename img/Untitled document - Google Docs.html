HTML 5 
HTML stands for HyperText Markup Language, HTML5 is the major revision of HTML, earlier 
versions such as HTML4. HTML 5 comes with better features which include: 
●New Doctype:  It was impossible to master the XHTML document. HTML5 new 
doctype is short precise and one can easily master.​ <!DOCTYPE html> 
●Video and Audio Support : ​Unlike ​earlier versions of HTML which used third-party 
add-on or plugin for displaying media files, HTML5 offers native support using the two 
two elements, <audio> and <video>, for embedding audio and video files into a web page  
●New Input Types:  One can use the new input types which include: ​number,date 
/datetime, color, range, url, ema​il, tel. 
●Better Form Features: ​HTML5 offers better form features that can be used to check 
input for example: entering an email address would simply look like this on a form: 
○<input name="name" type="email" placeholder="EMAIL"> 
Specifying type as email ensures that whatever is entered has an email format when the 
submit button is pressed. 
●Custom Data attributes: ​HTML5 Custom Data Attributes (data-*) allow you to 
define custom attributes for an HTML element.  
●HTML5 can be used to develop mobile applications in both ISO and Android. 
 
 
NEW FEATURES IN HTML5 
New Elements 
Below is a few new elements in HTML5: 
●<header> </header>: ​Defines the header part of a page 
●<footer></footer>​Defines the footer part of a page 
●<section></section>: ​Defines a section in a page. 
●<aside></aside>:​Defines the content aside from the page; 
●<time>:​Defines time 
New APIs 
HTML5 API:​ It comes with APIs that support various features these include 
●HTML Geolocation:uses the ​getCurrentPosition() to get the location of a user. 
●HTML Local Storage: there are two storage types:  
○window.localStorage - long term storage 
○window.sessionStorage -shortterm-stores data for a session 
●HTML App Cache to enable caching just add the following to the code: 
○<html manifest="demo.appcache"> 
●HTML SSE. 
●HTML Drag/Drop: This feature allows draging of items and placing them at specific 
locations
